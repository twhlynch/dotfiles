# MacOS dotfiles

Run `SETUP.sh` to setup config symlinks.
Run `INSTALL.sh` to install brew and many packages.

> `chmod +x FILE.sh`

> vscode custom css and bg are not symlinked, just referenced, so make sure to clone `dotfiles` into `~`.
